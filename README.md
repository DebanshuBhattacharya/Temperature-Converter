#  Temperature Converter

A modern and interactive **Temperature Converter** built using HTML , CSS, JavaScript, and GSAP animations. The application provides instant temperature conversions with smooth animations, clean UI, and real-time validation.





##  Features

- 🌡️ Convert between:
  - Celsius → Fahrenheit
  - Fahrenheit → Celsius
  - Celsius → Kelvin
  - Kelvin → Celsius
- ⚡ Instant conversion
- 🎨 Modern glassmorphism-inspired UI
- ✨ Smooth GSAP animations
- 🔥 Floating background effects
- 📱 Responsive design
- 🔄 One-click reset
- ❌ Input validation with animated error messages
- 📏 Automatic result formatting
- 🧮 Displays conversion formula

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- GSAP (GreenSock Animation Platform)

---



---


## ⚠️ Validation

The application prevents invalid inputs such as:

- Empty values
- Non-numeric input
- Kelvin below **0 K**
- Celsius below **−273.15°C (Absolute Zero)**

---

## 🎬 Animations

This project uses **GSAP** to create:

- Card entrance animation
- Floating background symbols
- Hover effects
- Result transitions
- Reset animation
- Input shake on error
- Badge transitions
